Help

Display most interesting photographs that people on the Flickr photo-sharing web site have taken close to your current location. Swipe left and right to browse through the photographs. See new photos loaded as you travel about. See the position of the photo on a map or go to its Flickr web page to see more. (All photographs are copyright of their owners.)

controls

At the main photo display view: flick sideways to see more photos, tap the full-screen button to hide the visible controls, tap the "i" button to see more information about the photo. Rotate the phone sideways if necessary to get photographs to better fill the screen.

When viewing the information about the photo you can tap the Flickr icon to view on the Flickr photo-sharing or press and hold on the map to bring up the Google Map app in a new card.

As with most WebOS apps you can get back to the previous view by making the back gesture right-to-left on the blank area below the screen.

To customize the app, choose "Preferences" from the small menu at the top left of the screen.

How photos are found

The application queries Flickr to return the most "interesting" photos in a given radius around your current position. (Where "Interestingness" is a value that Flickr calculates for each photograph based on many different factors.) It does not know ahead of time the local density of photographs, so it repeatedly tries querying with different search radiuses until it gets backs a reasonable number of photographs. It shows the most interesting photograph initially, and as you flick left and right away from the initial photograph you get next most interesting photographs.

In the application menu, under "Preferences", you can switch the application to ask Flickr for the most recent photographs, rather than the most interesting.

The application will only do a new query with a new search radius when you have not touched the application for a little while, so try not touching the phone for about ten seconds to see if you get some new photos.

Privacy Statement

This application does not store any permanent information either on your phone or on the Internet. It is registered with Flickr which does gather anonymous aggregate counts of the connections the application makes to the Flickr photo-sharing web site.

The source code is on Github from where you can check it out using the git source-code control program. It comes with a Makefile that builds and installs it on Linux and Mac.
